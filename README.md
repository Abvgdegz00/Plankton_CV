# Описание проекта

Проект посвящён классификации микроскопических изображений планктона, принадлежащих трём различным видам. Датасет ограничен по объёму и несбалансирован по классам, что требует использования стратегий transfer learning и устойчивых метрик для корректной оценки моделей.

# Цель проекта:
Исследовать различные архитектуры и стратегии обучения нейронных сетей, а также провести сравнительный анализ их эффективности.

# Используемые модели

* **PlanktonNet** - собственная сверточная сеть, обучаемая с нуля.

* **ConvNeXt-Tiny** - предобученная архитектура с тремя стратегиями fine-tuning:

    * обучение только классификатора

    * частичное дообучение

    * полное дообучение

* **EfficientNet-B4** - предобученная архитектура с аналогичными стратегиями.
